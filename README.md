# SoulTypes MBTI

MBTI 유형을 입력하거나 선택하면 주요 특징, 영성적 특징, 소외되기 쉬운 환경, 추천 Soulwork를 보여주는 모바일 최적화 정적 웹앱입니다.

## 기능

- 4개 질문으로 E/I, S/N, T/F, J/P 선호를 가볍게 진단합니다.
- 약식 결과를 기존 16유형 영성 해석 카드와 자동으로 연결합니다.
- 직접 유형을 입력하거나 16유형 버튼에서 선택할 수도 있습니다.

## 자료 반영

- `SoulTypes - 통합본.pdf`의 핵심 관점인 "성격 유형은 영적 정답이 아니라 자연스러운 경로를 찾는 지도"를 앱 구조에 반영했습니다.
- `soultype.txt`의 기능별 영적 경로 정리도 보조 자료로 확인해 8가지 주기능 경로를 결과 카드에 추가했습니다.
- 앱 문장은 원문 복사가 아니라 일반 MBTI 지식과 영성 성찰 관점을 바탕으로 새로 작성했습니다.
- 앱 하단에 참고 도서와 사용자 제공 자료 출처를 표시했습니다.
